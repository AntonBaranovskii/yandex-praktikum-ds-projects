# Прогнозирование заказов такси

Задача - построить модель предсказания количества заказов такси на следующий час по историческим данным.

## Данные

Признаки

- datetime - дата/время
- num_orders - количество заказов

## Используемые библиотеки

- *pandas*
- *sklearn*
- *matplotlib*
- *catboost*
- *lightgbm*
